# 07 — Технологічний стек

Чесний поділ: що **реально в роботі** vs що **лише заплановано** в первісному баченні.

## Backend

| Технологія | Статус | Призначення |
|---|---|---|
| Python 3.12+ | ✅ | Мова бекенду |
| FastAPI | ✅ | HTTP API, async |
| Pydantic | ✅ | Валідація запитів (`ChatRequest`) |
| asyncio | ✅ | Паралельні запити до моделей |
| `groq` SDK | ✅ | Провайдер Groq |
| `google-generativeai` | ✅ | Провайдер Gemini (суддя) |
| Cerebras / SambaNova SDK | ✅ | Провайдери-відповідачі |
| Python generators + `StreamingResponse` | ✅ | Стрімінг |
| `python-dotenv` | ✅ | `.env` |
| **LiteLLM** | ⛔ | Планувалось як єдиний шар провайдерів — **не використовується** (прямі SDK) |
| **httpx** | ⛔ | Планувалось для async-запитів — **не використовується** |
| **SQLAlchemy** | ⛔ | ORM — не використовується |
| **SQLite / PostgreSQL** | ⛔ | БД — немає (стан in-memory) |
| **ChromaDB** | ⛔ | Vector DB для RAG — немає |
| **PyPDF** | ⛔ | Читання PDF — немає |
| **PyYAML** | ⛔ | `prompts.yaml` — немає (промпт у коді) |
| **Pytest** | ⛔ | Тести — у репозиторії відсутні |

## Frontend

| Технологія | Статус | Призначення |
|---|---|---|
| Next.js (App Router) | ✅ | Каркас фронтенду |
| React | ✅ | UI |
| Tailwind CSS | ✅ | Стилі |
| Design tokens (`theme/*.ts`) | ✅ | Кольори/відступи/типографіка |
| i18n (uk/pl/en, JSON) | ✅ | Локалізація |
| TypeScript | 🟡 | Частково: `theme/`, `types/` — `.ts`; компоненти/сервіси — `.js/.jsx` |

## Infra / Deployment

| Технологія | Статус |
|---|---|
| Docker / Docker Compose | ⛔ немає у репозиторії |
| Ubuntu VPS | ⛔ |
| Nginx (reverse proxy) | ⛔ |
| SSL / Let's Encrypt | ⛔ |

## Висновок

Реальний стек **значно вужчий** за первісно задекларований. Це нормально для поточної фази, але документація мусить це відображати чесно: усі ⛔ — це **бачення/беклог**, а не наявні можливості. Пріоритети введення — у [09-roadmap.md](09-roadmap.md).
